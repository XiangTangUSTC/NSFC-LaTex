## 🚀 使用方法

### 编译要求

- **推荐编译器**：XeLaTeX
- **平台支持**：Windows、macOS、Linux
- 支持overleaf

### 编译步骤

```bash
# 使用 XeLaTeX 编译
xelatex 面上.tex
bibtex 面上
xelatex 面上.tex
xelatex 面上.tex
```
